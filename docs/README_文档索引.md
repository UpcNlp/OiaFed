# 📚 FedCL 项目完整文档索引

## 🎯 文档概览

本文档集合全面介绍了 **FedCL (Federated Continual Learning)** 框架的架构设计、组件实现和使用方法。FedCL 是一个专为联邦持续学习研究设计的框架，采用五层架构和装饰器驱动的设计理念。
